Project Description:
A simple project that uses express and mongodb to register & login to your account. This covers with authentication,sessions,bcrypt,jade etc. While registration password will be encrypted and stored in db along with http headers to avoid csrf attacks. While login it authenticates with email and password to log into your account. This and all done with views in jade,express with node and mongodb alone. 

To run this nodecase study you require some node_modules and mongodb

First install mongodb and set path for your db
run mongod
then run mongo

Install
npm install jade,express,mongoose,body-parser,client-seesions,bcrypt,csurf etc
